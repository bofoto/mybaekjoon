# 백준 문제 풀이

백준 온라인 저지 문제 풀이를 기록하는 저장소입니다.

## 사용 언어
- Java (주)
- Python (선택적)

## 폴더 구조
```
문제번호/
├── Main.java
└── main.py (선택적)
```

각 폴더는 백준 문제 번호로 구성되어 있으며, Java로 필수 풀이하고 일부 문제는 Python으로도 추가 풀이합니다.
